# PySprint Studio

PySprint Studio is a frontend-only Python practice platform with structured questions, hints, solutions, saved questions, notes, progress tracking, and JSON progress import/export.

## Run Locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 4174 --bind 127.0.0.1
```

Then visit:

```text
http://127.0.0.1:4174/
```

## Deploy

This project is static HTML/CSS/JavaScript, so it can be deployed to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host.

For GitHub Pages:

1. Push this folder to a GitHub repository.
2. In GitHub, open repository Settings.
3. Go to Pages.
4. Deploy from the `main` branch root.
